VARIÁVEIS EM CSS

COMO FAZER VARIÁVEIS

Para fazer uma variável em CSS você deve usar no seu < style > uma regra chamada de :root.

O que é :root ?

:root é uma pseudoclasse das CSS que imdica a raiz do seu site, ou seja, toda regra apropriada nela se torna uma raiz que você pode usar, mudar e estilizar quando quiser dês que você determine.

Use dentro do :root (--)

(--) indica que é uma variável
Exemplo:

:root { --cor-verde: #00000; }

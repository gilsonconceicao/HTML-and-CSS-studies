Tabelas no HTML

Como faz tabelas? É simples, as tabelas possuem 4 tegs, table, tr, th e td.

< Table >

O table funciona Para que você possa marcar a sua tabela, funciona como um agrupamento em bloco. Assim como div, header... entre outros. Claro, o table é especifica e apropriada para tabelas.

< Tr >

tr funciona como uma linha. O que será adicionado ali exemplo:

tr > Para titulos em vertical. tr > Para conteúdos.

< Th >

th especifico para colocar titulos da tabela, exemplo:

th > Pode ser em linha, assim você usaria com um tr apenas para títulos.

< Td >

td serve para colocar um cero valor na tabela, como: números, valores, média, resultados entre outras
